# ML4ES (2025): Лекция 4

## Постановка задач машинного обучения в науках о Земле

## 1. Введение
- Контекст лекции: разбор структуры домашнего задания №1.
- Цели занятия: понять, как формулировать ML‑задачи строго и корректно.
- Почему постановка задачи важнее, чем выбор модели.

## 2. Общая структура постановки задачи
1) Класс задачи (supervised / unsupervised / self-supervised / semi-supervised).  
2) Вид задачи (регрессия, классификация, кластеризация, аппроксимация распределения и др.).  
3) Целевая переменная:  
   - что вычисляем;  
   - тип (действительное число, вектор, категориальная метка);  
   - размерность.  
4) Объекты/события: что является элементами датасета.  
5) Признаковое описание:  
   - признаки, их физический смысл;  
   - тип каждого признака;  
   - размерность признакового вектора;  
   - возможные дополнительные признаки;  
   - опционально — снижение размерности.  
6) Функция потерь.  
7) Мера качества:  
   - регрессия: RMSE, MAE, корреляция;  
   - классификация: accuracy, F1, доля верных ответов;  
   - неконтролируемые задачи: критерий, согласованный с экспертом.  

## 3. Пример 1: Даунскейлинг полей атмосферы
- Контекст задачи: восстановление высокодетальных полей из низкого разрешения.  
- Класс задачи: supervised learning.  
- Вид задачи: регрессия.  
- Целевая переменная: поле высокой размерности (действительный вектор ~10^6 значений).  
- Признаковое описание: поле низкого разрешения (+ дополнительные переменные).  
- Особенности:  
  - размерность признаков очень велика;  
  - координаты не включаются как признаки (константные значения).  
- Мера качества: RMSE между высокоразрешенными полями.

## 4. Пример 2: Обнаружение мезомасштабных конвективных систем (МКС)
- Данные: спутниковые каналы (несколько спектральных диапазонов).  
- Цель: определить положение и размеры МКС.  
- Класс задачи: supervised learning.  
- Вид: регрессия или detection‑задача.  
- Целевая переменная: координаты центра + размеры (4–5 чисел).  
- Признаковое описание: многоканальный спутниковый снимок.  
- Объект: фиксированная область пространства, наблюдаемая в один момент времени.  
- Мера качества: степень совпадения с экспертной разметкой.

## 5. Пример 3: Определение характеристик волнения по судовому радару
- Данные: неотфильтрованные радарные изображения (матрица 4096×4096).  
- Цели: значимая высота, длина и период волны.  
- Класс: supervised learning.  
- Вид: регрессия.  
- Целевая переменная: трехмерный действительный вектор.  
- Признаковое описание: амплитуда обратного рассеяния в каждой точке (≈16 млн значений).  
- Особенности: экстремальная размерность признаков → необходимость снижения размерности.  
- Мера качества: RMSE для каждой характеристики.

## 6. Пример 4: Кластеризация стратосферного полярного вихря
- Цели: определить типичные состояния вихря.  
- Класс: unsupervised learning.  
- Вид: кластеризация.  
- Признаковое описание: поля геопотенциала на уровне 10 гПа.  
- Целевая переменная отсутствует (метка кластера определяется алгоритмом).  
- Оценка качества: согласование результатов с экспертными календарями SSW.

## 7. Общие рекомендации для домашнего задания
- Не пытаться решать задачу — только формулировать.  
- Формулировать строго, избегать расплывчатых понятий (“далеко”, “близко”).  
- Всегда давать численные описания целевой переменной.  
- Включать мотивацию постановки (2–3 предложения).  
- Писать много слов: объяснять каждое решение, каждую формулировку.  
- В Jupyter‑ноутбуке использовать Markdown‑ячейки, формулы, текстовые пояснения.
