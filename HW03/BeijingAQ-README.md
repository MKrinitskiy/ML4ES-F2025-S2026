## Коллекция данных **Beijing Air Quality & Weather**

\* Набор данных наблюдений за качеством воздуха и погодными условиями в Пекине (Китай).

------

### Ссылки на данные и метаданные

- **Основной архив:** *Kaggle – Beijing Multi-Site Air-Quality Data Set (2010–2014)* — https://www.kaggle.com/datasets/sid321axn/beijing-multisite-airquality-data-set
- **Первоначальный источник:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/datasets/Beijing+PM2.5+Data)

------

### Формат и структура файлов

- Формат: `CSV`
- Разделитель полей: запятая `,`
- Десятичный разделитель: точка `.`
- Данные агрегированы по часу; каждая строка — одно часовое наблюдение.
- Доступны измерения для нескольких метеостанций Пекина (например, Aotizhongxin, Changping, Guanyuan, Wanshouxigong и др.).

------

### Переменные

| Имя столбца | Единицы | Описание                                                     |
| ----------- | ------- | ------------------------------------------------------------ |
| `No`        | —       | Порядковый номер записи                                      |
| `year`      | —       | Год наблюдения                                               |
| `month`     | —       | Месяц                                                        |
| `day`       | —       | День                                                         |
| `hour`      | —       | Час наблюдения (UTC+8)                                       |
| `pm2.5`     | µg/m³   | Концентрация взвешенных частиц диаметром менее 2.5 мкм       |
| `DEWP`      | °C      | Температура точки росы                                       |
| `TEMP`      | °C      | Температура воздуха                                          |
| `PRES`      | hPa     | Атмосферное давление на уровне станции                       |
| `cbwd`      | —       | Направление ветра (категориальное значение, например, NW, SE и т. д.) |
| `Iws`       | m/s     | Средняя скорость ветра (действительное, float)               |
| `Is`        | hr      | Накопленное количество часов снега (целое, int)              |
| `Ir`        | hr      | Накопленное количество часов с дождем (целое, int)           |

------

### Объем и охват

- **Период:** 2010–2014 (включительно)
- **Дискретность:** 1 час
- **Число записей:** ~420 000 строк (общее по всем станциям)
- **Покрытие:** несколько метеостанций в пределах Пекина и пригородов

------

### Качество данных и особенности

- **Пропуски:** имеются отсутствующие значения (`NA`) для отдельных станций и периодов, особенно в начале ряда или при сбоях оборудования.
- **Выбросы:** возможны экстремальные пики концентраций загрязняющих веществ (например, эпизоды сильного смога).
- **Сезонность:** выраженные годовые циклы загрязнения (зимой значения PM2.5 выше из-за отопления и неблагоприятной циркуляции, летом — ниже).
- **Гетерогенность:** данные получены с разных станций, характеристики которых могут немного различаться.
- **Метеофакторы:** значения температуры, влажности и ветра тесно связаны с динамикой загрязнения (например, низкий ветер и высокая влажность усиливают смоги).
